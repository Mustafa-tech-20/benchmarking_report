# Vehicle Development Agent

AI-powered vehicle development analysis and comparison tool.

## Setup

1. Copy `.env.example` to `.env` and fill in your credentials
2. Install dependencies: `pip install -r requirements.txt`
3. Run the backend: `python main.py`

### Frontend Setup

1. `cd frontend`
2. Copy `.env.example` to `.env`
3. Install: `npm install`
4. Run: `npm run dev`

## Features

- Compare up to 10 vehicles with 87 specifications
- Upload PDF/Excel files for spec extraction
- RAG integration for internal prototypes (CODE: prefix)
- Interactive HTML reports with charts
